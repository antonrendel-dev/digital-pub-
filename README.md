## О проекте

Сайт-агрегатор вакансий и резюме из Telegram-каналов. Контент автоматически подтягивается из публичных каналов, размечается тегами и публикуется по разделам. Пользователи могут самостоятельно подавать вакансии и резюме через форму — после модерации. Раздел статей для SEO-индексации.

## Структура проекта

```
.claude/                    # База знаний для AI-агентов
├── skills/
│   └── project-knowledge/  # Project docs (architecture, patterns, etc.)

app/                        # Next.js App Router страницы
components/                 # UI компоненты
lib/                        # Бизнес-логика (db, sync, auth)
prisma/                     # Схема БД и миграции
work/                       # Активные фичи
└── completed/              # Завершённые задачи
```

## Запуск локально

```bash
npm install
cp .env.example .env        # заполни переменные
npm run dev                 # http://localhost:3000
```

## Методология разработки

Проект использует **spec-driven подход** с AI-агентами:

1. **User Spec** → описываем ЧТО и ЗАЧЕМ
2. **Tech Spec** → описываем КАК реализовать
3. **Tasks** → декомпозиция на задачи
4. **Implementation** → AI-агент пишет код
